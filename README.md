# 简介
这个仓库是用来干什么的呢? 学习? 教学? 好像都不是. 最近新下载了很多东西, 有些流程比较简单, 而有些较为复杂. 除此之外呢, 还搞了搞驱动. 当前弄的时候, 可能会比较头疼, 经过一系列操作解决问题后, 也就享受于此. 可是, 解决这个问题的流程, 没有记录下来, 以后遇到相同的问题时, 没有任何参考, 仍需从头再来, 缺乏安全感.
所以, 这个仓库用来记录各类工具链的下载流程, 以及其中遇到的各种各样的问题的. 
格式是什么样的呢? 我也还没有想好. 想了想, 是必须要留下心理表征的, 因此需要准备很多个文件夹的, 好吧.
